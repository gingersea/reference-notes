# 中国史结构化总纲

## 总体框架

中国历史从约公元前2070年夏朝建立至公元1949年中华人民共和国成立，跨越约四千年（考古学上的文明起源可上溯至更早）。本系列将中国历史分为十五个时期，每期独立成篇，以年代、重大事件、管理方式为基本框架，最后以唯物史观视角进行综合分析。

## 六大历史脉络

### 一、政治制度的演进

从夏商的部落联盟式国家 → 西周封建宗法 → 秦汉中央集权郡县制 → 隋唐三省六部 → 宋二府三司 → 元行省 → 明废丞相设内阁 → 清军机处君主专制顶峰 → 民国共和尝试 → 人民共和国。

核心线索是**皇权与相权、中央与地方**两对矛盾的此消彼长。总体趋势是皇权不断加强（至清达顶峰），中央对地方的控制日益严密。

### 二、经济基础的变化

从井田制（周）→ 土地私有制确立（战国秦汉）→ 均田制（北魏至唐）→ 土地私有深化（宋以后"不立田制"）→ 近代经济转型。

赋税从劳役地租 → 实物地租 → 货币地租（一条鞭法、摊丁入亩），反映商品经济的发展。

### 三、选官制度的变迁

世卿世禄（夏商周）→ 军功爵制（秦）→ 察举征辟（汉）→ 九品中正（魏晋）→ 科举制（隋至清）→ 现代教育选官（民国至今）。

从"血统决定"到"考试选拔"，是社会流动性的逐步扩大。

### 四、思想文化的流变

巫史文化（商）→ 礼乐文化（周）→ 百家争鸣（春秋战国）→ 儒学独尊（汉）→ 玄学佛学兴起（魏晋）→ 三教并立（唐）→ 理学（宋明）→ 考据学（清）→ 新文化运动与马克思主义（近现代）。

### 五、民族关系的格局

华夏族（汉族前身）在黄河流域形成 → 春秋战国夷夏之辨 → 秦汉统一后汉族形成 → 魏晋南北朝民族大融合 → 隋唐胡汉一体 → 宋辽金夏多元对峙 → 元清少数民族统一王朝 → 现代多民族国家。

中国历史始终是**多民族互动融合**的过程，汉族本身就是长期民族融合的产物。

### 六、对外关系的轨迹

先秦的封闭发展 → 汉代丝绸之路开辟 → 唐代全面开放 → 宋元海上贸易繁荣 → 明代海禁与郑和下西洋 → 清代闭关与被迫开放 → 近现代融入世界体系。

从"天下"观念到"民族国家"观念的转变是中国对外关系史上最深刻的变革。

## 阅读建议

1. 按时间顺序阅读各朝代文档，形成基本的历史时序感
2. 重点关注**制度连续性**——许多"创新"实为前朝实验的完善
3. 注意**区域差异**——中国幅员辽阔，各区域发展不平衡是常态
4. 结合唯物史观解读，理解历史发展背后的物质动因
